Statistics

Statistics is the branch of mathematics dealing with data collection, analysis, interpretation, presentation, and organization.

Basic Concepts

Mean (Average)

The sum of data values divided by the number of values.

Median

The middle value when data is arranged in order.

Mode

The value that appears most frequently in the data set.

Range

Difference between the highest and lowest data values.

Graph Examples

Bar Chart: Shows the frequency of categories.

Pie Chart: Displays proportions of a whole.

Line Chart: Shows data trends over time.

Step-by-Step: Calculating Mean

  1. Sum all the data values.
  2. Count the number of values.
  3. Divide the sum by the count.
  4. The result is the mean (average).

Practice Problems

1. Find the mean of the data set: 3, 7, 7, 10, 12.

Solution:

Sum = 3 + 7 + 7 + 10 + 12 = 39. Number of values = 5. Mean = 39 / 5 = 7.8.

2. Identify the mode of the data set: 4, 4, 5, 6, 6, 6, 7.

Solution:

Mode is 6 because it appears most frequently.

3. Calculate the range of the data set: 15, 22, 8, 17, 10.

Solution:

Range = Highest value - Lowest value = 22 - 8 = 14.
