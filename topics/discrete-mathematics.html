Discrete Mathematics

Discrete Mathematics is the study of mathematical structures that are fundamentally discrete rather than continuous. It includes topics such as logic, set theory, combinatorics, graph theory, and algorithms.

Basic Concepts

Set Theory

A set is a collection of distinct objects, considered as an object in its own right. Sets are usually denoted by curly braces, e.g., {a, b, c}.

Logic

Logic deals with reasoning and propositions. A proposition is a statement that is either true or false. Logical connectives include AND (∧), OR (∨), NOT (¬), and IMPLIES (→).

Functions and Relations

A function maps elements from one set to another. A relation defines a relationship between elements of sets.

Graph Theory

Graphs are collections of nodes (vertices) connected by edges. They can represent networks, paths, and connections.

Example 1: Determine if the set {1, 2, 3, 4} is a subset of {1, 2, 3, 4, 5, 6}.

Solution: Since every element of the first set is in the second, {1, 2, 3, 4} \subseteq {1, 2, 3, 4, 5, 6}.

Example 2: Evaluate the logical expression (p \wedge \neg q) \lor r when p = true, q = false, and r = false.

Solution: \neg q = true, so p ∧ ¬q = true ∧ true = true, and true ∨ false = true.

Step-by-Step Methods

Determining Subsets

  1. Check each element of the first set.
  2. Verify if it is present in the second set.
  3. If all elements are present, the first set is a subset of the second.

Evaluating Logical Expressions

  1. Assign truth values to variables.
  2. Apply NOT operations.
  3. Evaluate AND and OR operations following precedence.
  4. Determine the final truth value.

Practice Problems

1. Is the set {2, 4} a subset of {1, 2, 3, 4, 5}?

Solution:

Yes, since both 2 and 4 are elements of the second set.

2. Evaluate p \lor (q \wedge \neg r) given p = false, q = true, and r = true.

Solution:

\neg r = false, so q ∧ ¬r = true ∧ false = false, and p ∨ false = false.

3. Find the complement of the set {1, 3, 5} assuming the universal set is {1, 2, 3, 4, 5, 6}.

Solution:

The complement is all elements in the universal set not in the given set: {2, 4, 6}.

4. In graph theory, how many edges does a complete graph with 5 vertices have?

Solution:

A complete graph with n vertices has \frac{n(n-1)}{2} edges. For n=5, edges = \frac{5 \times 4}{2} = 10.
