Numbers

Numbers are the basic building blocks of mathematics, used to count, measure, and label. They can be classified into various types based on their properties.

Types of Numbers

Natural Numbers

Counting numbers starting from 1, 2, 3, ...

Whole Numbers

Natural numbers including zero: 0, 1, 2, 3, ...

Integers

Whole numbers and their negatives: ..., -3, -2, -1, 0, 1, 2, 3, ...

Rational Numbers

Numbers that can be expressed as a fraction of two integers, e.g., 3/4, -5/2.

Irrational Numbers

Numbers that cannot be expressed as fractions, e.g., π, √2.

Real Numbers

All rational and irrational numbers combined.

Properties of Numbers

  • Commutative Property: a + b = b + a and ab = ba
  • Associative Property: (a + b) + c = a + (b + c) and (ab)c = a(bc)
  • Distributive Property: a(b + c) = ab + ac
  • Identity Elements: 0 for addition, 1 for multiplication

Examples

Example 1: Identify the type of number: -7

Solution: -7 is an integer, but not a natural or whole number.

Example 2: Is 0.75 rational or irrational?

Solution: 0.75 = 3/4, so it is a rational number.

Step-by-Step: Classifying Numbers

  1. Check if the number is positive and whole (natural number).
  2. If zero included, it’s a whole number.
  3. If negative whole number, it’s an integer.
  4. If decimal or fraction that can be written as ratio of integers, it’s rational.
  5. If decimal that cannot be expressed as a ratio, it’s irrational.

Practice Problems

1. Classify the number -3.5.

Solution:

-3.5 is a rational number (expressible as -7/2), but not an integer.

2. Is √16 a rational or irrational number?

Solution:

√16 = 4, which is a rational number.

3. Determine whether π is rational or irrational.

Solution:

π is an irrational number since it cannot be expressed as a fraction.

4. Is 0 a natural number?

Solution:

No, 0 is not considered a natural number, but it is a whole number.
