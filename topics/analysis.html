Introduction to Real Analysis

Real analysis is the rigorous study of limits, continuity, sequences, and functions of real numbers.

Fundamental Concepts

Limits

The limit of a function f(x) as x approaches a is the value that f(x) approaches as x gets arbitrarily close to a.

Continuity

A function f is continuous at x = a if \lim_{x \to a} f(x) = f(a).

Sequences

A sequence is an ordered list of numbers. A sequence {a_n} converges to a limit L if for every \epsilon > 0, there exists an N such that |a_n - L| < \epsilon for all n > N.

Series and Convergence

Series

A series is the sum of the terms of a sequence. The series \sum_{n=1}^\infty a_n converges if the sequence of partial sums converges.

Absolute and Conditional Convergence

A series is absolutely convergent if \sum |a_n| converges. If it converges but not absolutely, it is conditionally convergent.

Example: p-Series

The p-series \sum_{n=1}^\infty \frac{1}{n^p} converges if p > 1, and diverges otherwise.

Continuity and Limits

Properties of Continuous Functions

Continuous functions on closed intervals are bounded and attain their bounds (Extreme Value Theorem).

Intermediate Value Theorem

If a function f is continuous on [a, b] and L is any value between f(a) and f(b), then there exists c \in [a,b] such that f(c) = L.

Differentiation

Definition of the Derivative

The derivative of f at x=a is defined as f'(a) = \lim_{h \to 0} \frac{f(a+h) - f(a)}{h}, if this limit exists.

Mean Value Theorem

If f is continuous on [a,b] and differentiable on (a,b), there exists c \in (a,b) such that f'(c) = \frac{f(b)-f(a)}{b-a}.

Integration

Riemann Integral

A function f is Riemann integrable on [a,b] if the limit of Riemann sums exists as the partition gets finer.

Fundamental Theorem of Calculus

Relates differentiation and integration: If F is an antiderivative of f, then \int_a^b f(x) dx = F(b) - F(a).

Step-by-step Examples

Example 1: Find \lim_{x \to 2} (3x + 4).

  1. Substitute the value x=2 into the function: 3(2) + 4 = 6 + 4.
  2. Simplify: 10.
  3. Therefore, \lim_{x \to 2} (3x+4) = 10.

Example 2: Determine if the function f(x) = \frac{1}{x} is continuous at x=0.

  1. Check if f(0) exists: It does not, since division by zero is undefined.
  2. Therefore, f(x) is not continuous at x=0.

Example 3: Find the derivative of f(x) = x^2 at x=3.

  1. Apply the definition: f'(3) = \lim_{h \to 0} \frac{(3+h)^2 - 3^2}{h}.
  2. Expand numerator: (9 + 6h + h^2) - 9 = 6h + h^2.
  3. Simplify fraction: \frac{6h + h^2}{h} = 6 + h.
  4. Take limit as h \to 0: 6.
  5. Thus, f'(3) = 6.

Practice Problems

1. Evaluate \lim_{x \to 1} (2x^2 + 3x - 5).

Substitute x=1: 2(1)^2 + 3(1) - 5 = 2 + 3 - 5 = 0.

Answer: 0

2. Determine if f(x) = \frac{1}{x-2} is continuous at x=2.

f(2) is undefined because division by zero is undefined.

Therefore, f(x) is not continuous at x=2.
