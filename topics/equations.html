Equations

An equation is a mathematical statement asserting the equality of two expressions. Solving equations means finding the value(s) of the variable(s) that satisfy the equality.

Basic Concepts

Linear Equations

Linear equations are algebraic equations in which each term is either a constant or the product of a constant and a single variable. The general form is ax + b = 0, where a \neq 0.

Quadratic Equations

Quadratic equations have the form ax^2 + bx + c = 0, where a \neq 0. Solutions can be found using the quadratic formula, factoring, or completing the square.

Higher-Degree Equations

Equations of degree three or higher can be solved by factoring, synthetic division, or numerical methods, depending on the complexity.

Example 1: Solve the linear equation 3x - 7 = 2.

Solution: Add 7 to both sides: 3x = 9, then divide by 3: x = 3.

Example 2: Solve the quadratic equation x^2 - 5x + 6 = 0 by factoring.

Solution: Factor as (x - 2)(x - 3) = 0, so x = 2 or x = 3.

Step-by-Step Methods

Solving Linear Equations

  1. Isolate the variable term on one side.
  2. Perform inverse operations to simplify.
  3. Divide by the coefficient to solve for the variable.

Solving Quadratic Equations by Factoring

  1. Rewrite the equation in standard form.
  2. Factor the quadratic expression.
  3. Set each factor equal to zero.
  4. Solve each resulting linear equation.

Using the Quadratic Formula

  1. Identify coefficients a, b, and c.
  2. Plug into formula: x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}.
  3. Calculate the discriminant b^2 - 4ac.
  4. Compute the roots based on the discriminant value.

Practice Problems

1. Solve the equation 4x + 5 = 17.

Solution:

Subtract 5: 4x = 12. Divide by 4: x = 3.

2. Solve the quadratic equation x^2 + 4x + 3 = 0 by factoring.

Solution:

Factor as (x + 1)(x + 3) = 0, so x = -1 or x = -3.

3. Use the quadratic formula to solve 2x^2 - 4x - 6 = 0.

Solution:

Here, a=2, b=-4, c=-6.

Calculate discriminant: \Delta = (-4)^2 - 4 \times 2 \times (-6) = 16 + 48 = 64.

Roots: x = \frac{4 \pm 8}{4}, so x=3 or x=-1.

4. Solve for x in the equation 5(x - 2) = 3x + 6.

Solution:

Expand: 5x - 10 = 3x + 6.

Bring variables to one side: 5x - 3x = 6 + 10 → 2x = 16.

Divide by 2: x = 8.
