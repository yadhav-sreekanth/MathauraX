Decimals

Decimals are numbers expressed in the base-10 system with a decimal point separating the integer part from the fractional part.

Basic Concepts

Decimal Point

The dot (.) that separates the whole number part from the fractional part of a decimal number.

Place Value

Each digit after the decimal point represents a fractional part: tenths, hundredths, thousandths, etc.

Types of Decimals

Decimals can be terminating (finite digits) or repeating (infinite repeating digits).

Operations with Decimals

Addition: Align decimal points and add as usual.

  • Example: 3.75 + 2.4 = 6.15

Subtraction: Align decimal points and subtract.

  • Example: 5.6 − 3.25 = 2.35

Multiplication: Multiply as whole numbers ignoring the decimal, then place decimal in result.

  • Example: 1.2 × 3.4 = 4.08

Division: Make divisor a whole number by multiplying numerator and denominator by the same power of 10, then divide.

  • Example: 4.5 ÷ 1.5 = 3

Step-by-Step: Multiplying Decimals

  1. Ignore decimal points and multiply as whole numbers.
  2. Count total decimal places in both numbers.
  3. Place decimal point in product so that it has the total decimal places counted.
  4. Write the final product.
