Euclid's Postulates and Common Notions (Geometry)

Five Postulates

  1. A straight line segment can be drawn joining any two points.
  2. A straight line segment can be extended indefinitely in a straight line.
  3. A circle can be drawn with any center and radius.
  4. All right angles are equal to one another.
  5. Through a point not on a given line, exactly one parallel line can be drawn (Parallel Postulate).

Common Notions (Axioms)

  1. Things that are equal to the same thing are equal to each other.
  2. If equals are added to equals, the wholes are equal.
  3. If equals are subtracted from equals, the remainders are equal.
  4. Things that coincide with one another are equal to one another.
  5. The whole is greater than the part.

Other Important Geometry Postulates & Axioms

  • Playfair's Axiom: Through a point not on a given line, exactly one parallel line can be drawn (alternative formulation of Euclid's 5th postulate).
  • Ruler Postulate: Points on a line can be paired with real numbers so that the distance between points is the absolute difference of the numbers.
  • Segment Addition Postulate: If B lies between A and C on a line segment, then AB + BC = AC.
  • Angle Addition Postulate: If point D lies inside angle ABC, then m‚à†ABD + m‚à†DBC = m‚à†ABC.

Non-Euclidean Geometry Axioms

Hyperbolic Geometry

The Parallel Postulate is replaced by: For a given line and a point not on it, there exist infinitely many lines through the point not intersecting the given line.

Elliptic Geometry

No parallel lines exist; any two lines intersect.

Peano Axioms (Natural Numbers)

  1. 0 is a natural number.
  2. Every natural number has a unique successor.
  3. 0 is not the successor of any natural number.
  4. If the successor of a number equals the successor of another, then the numbers are equal.
  5. If a set contains 0 and is closed under the successor operation, then it contains all natural numbers (induction axiom).

Zermelo-Fraenkel Set Theory with Choice (ZFC)

ZFC axioms form the foundation of modern set theory.

  1. Axiom of Extensionality: Two sets are equal if and only if they have the same elements.
  2. Axiom of Empty Set: There exists a set with no elements.
  3. Axiom of Pairing: For any two sets, there is a set that contains exactly these two sets.
  4. Axiom of Union: For any set of sets, there is a set that contains exactly the elements of those sets.
  5. Axiom of Infinity: There exists an infinite set.
  6. Axiom Schema of Replacement: The image of a set under any definable function is also a set.
  7. Axiom of Power Set: For any set, there exists a set of all its subsets.
  8. Axiom of Regularity (Foundation): Every non-empty set has an element disjoint from it.
  9. Axiom of Choice: For any set of non-empty sets, there exists a choice function selecting an element from each set.

Practice Problems

Problem 1: Using Euclid's postulates, prove that the base angles of an isosceles triangle are equal.

By Peano's third axiom, 0 is defined as not being the successor of any number, which is foundational for natural number construction.
